# MindField

MindField is a personality profiling app built with React. It helps users explore their traits through interactive questions and visual insights—like stepping into the landscape of your own mind.

## Services

- [alertmanager](https://alertmanager.aldous.info/): Prometheus Alertmanager for grouping, silencing and sending alerts.  
- [api](https://api.aldous.info/): Nest-based backend serving the MindField API.  
- [backup](https://backup.aldous.info/): Daily Postgres dump & retention cleanup.  
- [blackbox-exporter](https://blackbox-exporter.aldous.info/): Probes HTTP/TCP endpoints for uptime checks.  
- [caddy](https://caddy.aldous.info/): TLS-terminating reverse proxy for all web services.  
- [gitea](https://gitea.aldous.info/): Self-hosted Git service.  
- [gitea-db](https://gitea-db.aldous.info/): PostgreSQL store for Gitea.  
- [grafana](https://grafana.aldous.info/): Dashboard UI for Prometheus metrics.  
- [grapesjs](https://grapesjs.aldous.info/): Headless editor service for dynamic forms.  
- [hasura](https://hasura.aldous.info/): Instant GraphQL API on Postgres.  
- [jaeger](https://jaeger.aldous.info/): Distributed-tracing storage & UI.  
- [keycloak](https://keycloak.aldous.info/): OAuth2/OpenID Connect identity provider.  
- [kong](https://kong.aldous.info/): API gateway & rate-limiter.  
- [kong-database](https://kong-database.aldous.info/): PostgreSQL store for Kong.  
- [loki](https://loki.aldous.info/): Log aggregation compatible with Prometheus.  
- [mailhog](https://mailhog.aldous.info/): Fake SMTP server for dev/testing.  
- [minio](https://minio.aldous.info/): S3-compatible object storage.  
- [mkdocs](https://docs.aldous.info/): Serves your Markdown docs (this site).  
- [node-exporter](https://node-exporter.aldous.info/): Exposes host OS metrics to Prometheus.  
- [opensearch](https://opensearch.aldous.info/): Elasticsearch-compatible search & analytics.  
- [opensearch-dashboards](https://opensearch-dashboards.aldous.info/): UI for OpenSearch.  
- [otel-collector](https://otel-collector.aldous.info/): Central OpenTelemetry collector & exporter.  
- [pgadmin](https://pgadmin.aldous.info/): Web UI for managing Postgres.  
- [pgbouncer](https://pgbouncer.aldous.info/): Lightweight Postgres connection pooler.  
- [postgraphile](https://postgraphile.aldous.info/): Instant GraphQL server on Postgres.  
- [postgres](https://postgres.aldous.info/): Primary relational database.  
- [postgrest](https://postgrest.aldous.info/): REST API server for Postgres.  
- [prisma-studio](https://prisma-studio.aldous.info/): GUI for inspecting your Prisma schema & data.  
- [prometheus](https://prometheus.aldous.info/): Time-series database for metrics.  
- [promtail](https://promtail.aldous.info/): Ships logs to Loki.  
- [rabbitmq](https://rabbitmq.aldous.info/): Message broker for background jobs.  
- [redaction](https://redaction.aldous.info/): FastAPI-based PII redaction service.  
- [redis](https://redis.aldous.info/): In-memory data store & cache.  
- [redoc](https://redoc.aldous.info/): OpenAPI documentation UI.  
- [render](https://render.aldous.info/): Service that generates PDFs/previews.  
- [sonarqube](https://sonarqube.aldous.info/): Continuous code-quality analysis.  
- [sonarqube-db](https://sonarqube-db.aldous.info/): PostgreSQL for SonarQube.  
- [step-ca](https://step-ca.aldous.info/): Internal certificate authority.  
- [storybook](https://storybook.aldous.info/): React component explorer.  
- [submission](https://submission.aldous.info/): Microservice handling submissions.  
- [swagger-ui](https://swagger-ui.aldous.info/): Interactive API docs for `api`.  
- [transform](https://transform.aldous.info/): Microservice that sanitises & transforms payloads.  
- [trivy](https://trivy.aldous.info/): Container image security scanner.  
- [uptime-kuma](https://uptime-kuma.aldous.info/): Self-hosted uptime monitoring dashboard.  
- [web](https://aldous.info/): Next.js frontend for MindField.
