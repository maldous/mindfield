# MindField

MindField is a personality profiling app built with React. It helps users explore their traits through interactive questions and visual insights—like stepping into the landscape of your own mind.

## Services

- **alertmanager**: Prometheus Alertmanager for grouping, silencing and sending alerts.
- **api**: Nest-based backend serving the MindField API.
- **backup**: Daily Postgres dump & retention cleanup.
- **blackbox-exporter**: Probes HTTP/TCP endpoints for uptime checks.
- **caddy**: TLS-terminating reverse proxy for all web services.
- **gitea**: Self-hosted Git service.
- **gitea-db**: PostgreSQL store for Gitea.
- **grafana**: Dashboard UI for Prometheus metrics.
- **grapesjs**: Headless editor service for dynamic forms.
- **hasura**: Instant GraphQL API on Postgres.
- **jaeger**: Distributed-tracing storage & UI.
- **keycloak**: OAuth2/OpenID Connect identity provider.
- **kong**: API gateway & rate-limiter.
- **kong-database**: PostgreSQL store for Kong.
- **loki**: Log aggregation compatible with Prometheus.
- **mailhog**: Fake SMTP server for dev/testing.
- **minio**: S3-compatible object storage.
- **mkdocs**: Serves your Markdown docs (this site).
- **node-exporter**: Exposes host OS metrics to Prometheus.
- **opensearch**: Elasticsearch-compatible search & analytics.
- **opensearch-dashboards**: UI for OpenSearch.
- **otel-collector**: Central OpenTelemetry collector & exporter.
- **pgadmin**: Web UI for managing Postgres.
- **pgbouncer**: Lightweight Postgres connection pooler.
- **postgraphile**: Instant GraphQL server on Postgres.
- **postgres**: Primary relational database.
- **postgrest**: REST API server for Postgres.
- **prisma-studio**: GUI for inspecting your Prisma schema & data.
- **prometheus**: Time-series database for metrics.
- **promtail**: Ships logs to Loki.
- **rabbitmq**: Message broker for background jobs.
- **redaction**: FastAPI-based PII redaction service.
- **redis**: In-memory data store & cache.
- **redoc**: OpenAPI documentation UI.
- **render**: Service that generates PDFs/previews.
- **sonarqube**: Continuous code-quality analysis.
- **sonarqube-db**: PostgreSQL for SonarQube.
- **step-ca**: Internal certificate authority.
- **storybook**: React component explorer.
- **submission**: Microservice handling loan submissions.
- **swagger-ui**: Interactive API docs for `api`.
- **transform**: Microservice that sanitises & transforms payloads.
- **trivy**: Container image security scanner.
- **uptime-kuma**: Self-hosted uptime monitoring dashboard.
- **web**: Next.js frontend for MindField.
